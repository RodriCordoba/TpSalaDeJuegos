Mi Juego Propio: Simón

Implementé el clásico juego Simón para poner a prueba la memoria y los reflejos del jugador, siguiendo secuencias de colores que aumentan progresivamente su dificultad y con un tiempo límite para responder.

Mecánica del Juego:

  1. El juego muestra un color.
  2. El jugador debe repetir la selección.
  3. El juego repite la secuencia y añade un nuevo color.
  4. El jugador debe repetir la nueva secuencia completa.
  5. La secuencia se hace más larga con cada acierto.
¡Fin del Juego!
La partida termina si el jugador se equivoca de color o se acaba el tiempo para responder.
